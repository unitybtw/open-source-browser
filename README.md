# Nova Browser 🚀

A modern, AI-powered Electron + React browser built for speed, privacy, and ultimate productivity.

## ✨ Key Features
- **Virtual AI Cursor**: AI agents connecting via MCP can control the active tab with a magical glowing cursor, mimicking real user interactions.
- **Model Context Protocol (MCP)**: Native support for external AI assistants to read, click, and type directly into the current webpage.
- **Split View**: Seamlessly snap two tabs side-by-side with an adjustable resizing handle for maximum productivity.
- **Advanced Download Manager**: Pause, resume, and track downloads with an elegant UI and history persistence.
- **Privacy Shield & VPN**: Built-in ad blocker and proxy VPN capabilities for anonymous, tracker-free browsing.
- **Vertical Tabs & Workspaces**: Organize your workflow with color-coded workspaces and a sleek vertical tab sidebar.
- **Smart Omnibox**: Instantly search history, bookmarks, and open tabs with Spotlight-like speed.
